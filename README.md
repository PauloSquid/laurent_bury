# Site Vitrine Traducteur

Site vitrine pour un traducteur professionnel, développé avec Next.js, TypeScript et Tailwind CSS.

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur.

## Structure

- `/` - Page d'accueil avec message de bienvenue
- `/traduction` - Liste des traductions (tri par genre ou éditeur)
- `/cv-contact` - CV et informations de contact
- `/dernieres-parutions` - Dernières parutions
- `/travaux-presse` - Travaux de presse

## Technologies

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
